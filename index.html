چت از طریق GitHub Actions؛ مناسب وقتی فقط github.com باز است. درخواست‌ها در repo عمومی به‌صورت رمزنگاری‌شده ذخیره می‌شوند.

مهم: GitHub Token فقط در مرورگر خودت ذخیره می‌شود. کلید مدل‌ها مثل OpenRouter/Groq/Puter/Gemini باید در GitHub Secrets باشند و هرگز از مرورگر ارسال نمی‌شوند.
GitHub Username
Repository Name
GitHub Fine-grained PAT - Contents Read/Write
Security Key / همان CHAT_QUEUE_KEY
برای repo عمومی این فیلد را خالی نگذار. اگر خالی باشد prompt و response عمومی می‌شود.
اجباری: در Settings → Secrets and variables → Actions این Secret را بساز: CHAT_QUEUE_KEY و مقدارش دقیقاً همان Security Key بالا باشد.
اختیاری، بسته به providerهایی که می‌خواهی استفاده کنی:
OPENROUTER_API_KEY برای OpenRouter
GROQ_API_KEY برای Groq
GEMINI_API_KEY برای Gemini مستقیم
XAI_API_KEY برای Grok مستقیم
PUTER_TOKEN برای Puter
برای GitHub Models معمولاً Secret جدا لازم نیست؛ همین GITHUB_TOKEN داخلی Action استفاده می‌شود. فقط workflow باید permission models: read داشته باشد.
